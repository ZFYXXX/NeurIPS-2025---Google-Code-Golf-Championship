# 🥉 Kaggle: NeurIPS 2025 - Google Code Golf Championship

This repository contains our **Bronze Medal** winning Python 3 solutions for the [NeurIPS 2025 - Google Code Golf Championship](https://www.kaggle.com/) hosted on Kaggle.

## 🏆 Competition Overview
The competition challenged participants to solve 400 tasks from the public training set of the **ARC-AGI benchmark suite** created by François Chollet. The ultimate goal was to produce functionally correct programs for each task using the **fewest possible number of characters** (Code Golf).

Despite the remarkable progress of state-of-the-art AI systems, they struggle with new problems beyond their training data. These extremely concise implementations emphasize robustness and simplicity, serving as canonical reference solutions for this seminal dataset, and contributing toward the development of more versatile AI systems.

## 📁 Repository Structure
- `task001.py` to `task400.py`: Our minimal Python 3 source codes for all 400 ARC-AGI tasks in the public training set (v1).
